Titik mulai kehabisan stock problem. Kali ini akan sangat mudah. Diberikan satu posisi awal si Titik yakni X,Y (-1000 <= X,Y <= 1000) pada bidang kertesius. Lalu string S (1 <= S <= 1000) yang hanya terdapat karakter 'V', '^', '<', dan '>'. Setiap karakter mendeskripsikan perpindahan pada bidang kartesius. 'V' menandakan perpindahan ke selatan, '<' menandakan perpindahan ke barat, '>' menandakan perpindahan ke timur, dan '^' menandakan perpindahan ke utara.

Tentukanlah posisi Titik setelah melakukan K langkah.

Input
Baris pertama dua bilangan integer X dan Y. Lalu pada baris kedua terdapat sebuah string S sesuai deskripsi. Lalu pada baris ketiga satu bilangan interger K (K <= S.length).

Output
Posisi Titik setelah melakukan K langkah dalam format X,Y.
